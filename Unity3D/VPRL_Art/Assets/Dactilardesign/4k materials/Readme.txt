I plan to add more materials and make it a paid pack sometime in the future, but for now, no cost.

This pack contains 38 PBR materials with texture sets sized 4096x4096. But I don't recommend to use 
all maps that size, i.e. you can lower some like the Height Map and achieve better results than full size
and save a lot of texture memory.
